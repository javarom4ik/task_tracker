Текущее состояние проекта

⚠️ Проект находится в разработке

На текущем этапе реализованы CRUD операции для управления задачами.

Реализованные функции:

✅ Create - Создание новых задач

✅ Read - Просмотр списка задач и детальной информации

✅ Update - Редактирование существующих задач

✅ Delete - Удаление задач

После запуска приложения вы можете:
- Создавать новые задачи
- Просматривать список всех задач
- Редактировать существующие задачи
- Удалять задачи

Планируемые улучшения
- Фильтрация и сортировка задач
- Категории и теги
- Напоминания и дедлайны
- Поиск по задачам
- Авторизация пользователей

Технический стек
Backend: Java 17, Spring Boot 3.5
Database: PosgreSQL
